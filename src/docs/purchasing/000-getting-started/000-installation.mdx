Walters & Wolf Office Add-ins have to be linked into your Excel installation to
work. The following steps are required to use the add-ins whenever you get a new
computer, re-install office, or after some office updates.

## Add the Walters & Wolf Add-in Catalog to Excel

1. Open Excel and if you have an file open, click into the **File** menu. If you
   don't have a file open, you can skip to the next step.
   <img
     alt="Excel File Menu"
     src="../../../ExcelRibbonFile.png"
     width="571"
     height="316"
   />
2. In the side menu, click on **Options** to show the Excel Options window.
   <img
     alt="Excel Options"
     src="../../../ExcelOptions.png"
     width="255"
     height="684"
   />
3. In the Options dialog, navigate to the **Trust Center** and then to
   **Trust Center Settings...**
   <img
     alt="Excel Trust Center Settings"
     src="../../../ExcelTrustCenterSettings.png"
     width="782"
     height="554"
   />
4. In the side menu, go to **Trusted Add-in Catalogs**. Enter in the value
   `\\wwdatahq\acad\Applications\Office` (copy and paste) and click on
   **Add catalog**
   <img
     alt="Excel Add Catalog"
     src="../../../ExcelAddCatalog.png"
     width="616"
     height="646"
   />
5. The catalog path should now be listed in the list box. Make sure the box is
   checked to **Show in Menu**, then exit out by clicking **OK**. You will get a
   message about needing to restart Excel to see the changes.
   <img
     alt="Excel Show Catalog in Menu"
     src="../../../ExcelShowCatalogInMenu.png"
     width="624"
     height="700"
   />
6. You have successfully installed the Add-in Catalog! Please restart Excel
   before trying to load the add-in into the ribbon.

## Load W&W Purchasing into the Ribbon

1. If you just added the Add-in Catalog to Excel, please restart Excel before
   continuing.
2. With any Excel file open, in the **Home** ribbon tab, click on the
   **Add-ins** button, then click **Advanced...** at the bottom of the pop-up
   <img
     alt="Excel Addin Advanced"
     src="../../../ExcelAddinAdvanced.png"
     width="1023"
     height="718"
   />
3. A dialog should appear with the **Walters & Wolf Purchasing Addins** listed.
   If it isn't there, repeat the steps to add the Add-in Catalog or make sure
   you have access to the internal network. Highlight the Purchasing Addin and
   click **Add**.
   <img
     alt="Excel Add Purchasing Addin"
     src="../../../ExcelAddPurchasingAddin.png"
     width="504"
     height="292"
   />
4. Verify the Add-in is loaded by checking that the ribbon now has the
   **W&W Purchasing** tab and has the **Vista** export button.
   <img
     alt="Excel Ribbon Purchasing Vista Export"
     src="../../../ExcelRibbonPurchasingVistaExport.png"
     width="1159"
     height="296"
   />
5. Congratulations, you are now ready to use the add-in!
